## Triết lý thiết kế item

### Tham khảo từ lore

**Phàm Nhân Tu Tiên:**
- Linh thảo được phân cấp theo tuổi (trăm năm → nghìn năm → vạn năm)
- Ví dụ lore: *Cuồng Diễm Thảo* (hỏa thuộc tính, vạn năm), *Cầu Long Thảo* (hình rồng), *Thiên Linh Trúc Diệp Thảo* (độc pháp tắc)
- Khoáng thạch: *Ngân Tâm Thạch*, *Huyền Thiết*, tinh quặng linh lực
- Luyện đan cần cả dược liệu lẫn khoáng vật làm phụ liệu

**Đấu Phá Thương Khung:**
- 10 phẩm đan dược, nguyên liệu cũng phân phẩm theo đó
- Dị hỏa làm xúc tác → concept "catalyst" item
- Năng lượng ba động khi phẩm cao → items tier cao có hiệu ứng "linh khí lan tỏa"

**Tiên Nghịch (Tiên Ni):**
- Items phân loại: Linh thảo / Khoáng thạch / Yêu thú vật liệu / Thiên địa linh vật
- Tinh thần chi lực → concepts liên quan đến soul/spirit materials
- Formations và seals là objects riêng (Cổ Truyền Tống Trận, Đại Na Di Lệnh)

### Nguyên tắc game design

1. **Tên phản ánh tier** — item tier 1 dùng từ "sơ khai", "phổ thông", "thô"; tier 5 dùng "tiên", "thần", "vạn linh"
2. **Tên phản ánh profession** — mỗi profession có "ngôn ngữ" riêng trong tên item:
   - Luyện Đan: thảo/hoa/quả/chi → dược liệu
   - Luyện Kim: khoáng/thiết/đồng/ngân/kim → kim loại
   - Phù Sư: linh sa/phù giấy/huyền mực/kim sa → vật liệu phù chú
   - Luyện Cổ Trùng: mật/nhưỡng/vỏ trùng/độc tinh → nguyên liệu nuôi cổ trùng
   - Thuần Thú: lông/lõi/vảy/nanh/huyết → vật liệu từ yêu thú (crafted = Linh Thú)
   - Trận Pháp: trận văn/linh sa/cổ trận → formation materials
3. **base_price thấp hơn nhiều so với gather fee** → tạo linh thạch sink (99.8% net loss)
4. **Crafted items = 10-50× nguyên liệu** → incentivize crafting

## Raw Materials — 50 Items (5 × 10 professions)

### Cấu trúc mỗi item

```
slug          : snake_case, unique identifier
nameVI        : tên tiếng Việt (theme tu tiên)
nameEN        : tên tiếng Anh (theme cultivation)
nameZH        : tên tiếng Trung
type          : itemTypeEnum value
tier          : 1-5 (Phase 02.1)
base_price    : linh thạch (rất thấp so với gather fee)
i18n_key      : game:items.raw.<slug>
lore_note     : mô tả ngắn (cho description i18n key sau)
```
